ShotClip

Mac screenshots — saved to a file and copied to the clipboard. At the same time, instantly.

macOS makes you pick: ⌘⇧4 saves a screenshot to a file, ⌃⌘⇧4 copies it to the clipboard — never both. So you either have a file you can't paste, or a paste with no file to keep.

⌘⇧4 → 📁 saved to your folder + 📋 on your clipboard

ShotClip removes the choice. Keep the native shortcuts you already know — every screenshot lands in your folder and is ready to ⌘V.

Download for macOS ★ Star on GitHub
brew install milhero/tap/shotclip

Free & open source (MIT) · macOS 13+ · ~300 KB · not notarized: first launch via right-click → Open

Why it feels instant

Two things usually make “auto-copy screenshot” tools feel laggy: Spotlight-based detection (1–2 s indexing delay) and macOS's floating thumbnail, which delays the file write by ~5 seconds. ShotClip watches the folder directly with a kqueue source and turns the thumbnail off by default — so the screenshot is on your clipboard the moment you release the mouse.

What it doesn't do

No annotation, no cloud upload, no scrolling capture, no subscription. If you want a full screenshot suite, look at CleanShot X or Shottr. ShotClip does the one thing macOS forgot, and stays out of the way — no new shortcuts, no screen-recording permission, no Electron.

Also in the menu bar

Change the system-wide screenshot folder, paste as pixels or as a file (both by default), clean the screenshot folder into the Trash with one click, launch at login.

Install

brew install milhero/tap/shotclip

Or download the zip and drop ShotClip.app into Applications. Or build from source:

git clone https://github.com/milhero/ShotClip.git
cd ShotClip && make install
